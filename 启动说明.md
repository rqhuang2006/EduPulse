# 启动说明

本项目已经内置 Python 3.14 运行时和运行依赖，评审机不需要提前安装 Python。
目前仅兼容Windows系统,macos不适用于该说明.
如遇问题,可先排查是否由总文件名过长/路径过长导致解压不完整,更换总文件名另存即可.

## 1. 推荐启动方式

在项目根目录执行：

```cmd
runtime\python.exe launch_all.py
```

启动流程会先启动后端 harness 服务，再启动前端 Streamlit 页面。



## 2. PowerShell 启动方式

在项目根目录执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\启动系统.ps1
```

只检查环境、不真正启动服务：

```powershell
powershell -ExecutionPolicy Bypass -File .\启动系统.ps1 -CheckOnly
```

## 3. 内置运行时目录

关键目录：

- `runtime\python.exe`：项目内置 Python 3.14.4
- `runtime\python314.zip`：Python 标准库
- `runtime\Lib\site-packages`：前后端所需第三方依赖

`launch_all.py` 也做了保护：即使用系统 Python 手动执行它，也会自动切换到 `runtime\python.exe`。

## 4. 默认地址

- 前端：`http://127.0.0.1:8501`
- 后端健康检查：`http://127.0.0.1:8000/health`

## 5. 日志位置

后端日志会写入：

- `.tmp\launcher_backend_stdout.log`
- `.tmp\launcher_backend_stderr.log`

关闭启动终端时，前后端会一起退出。
